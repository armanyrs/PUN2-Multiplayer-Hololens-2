# Proyek Akhir Workshop Teknologi MR

## Judul Proyek
**MR PC Assembly Guide with Voice Summon**

## Deskripsi Singkat
Proyek ini merupakan aplikasi Mixed Reality berbasis HoloLens yang bertujuan memberikan panduan interaktif dalam proses perakitan PC. Aplikasi ini dirancang untuk membantu pengguna belajar merakit komputer melalui simulasi visual 3D, panduan suara, serta fitur multiplayer untuk kolaborasi secara real-time.

## Fitur Utama
- **Voice Summon:** Panggil part menggunakan perintah suara
- **Multiplayer Mode:** Kolaborasi antar pengguna via Photon Unity Networking (PUN)
- **Rekomendasi Assemble Guide (RAG):** Sistem konsultasi otomatis berbasis voice-to-text
- **UI Interaktif dengan MRTK (Mixed Reality Toolkit)**

## Teknologi yang Digunakan
- Unity 2022.x
- MRTK 2.8.3
- HoloLens 2
- Photon Unity Networking (PUN)
- Visual Studio 2022
